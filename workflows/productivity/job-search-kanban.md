# Job Search Kanban

> Turn job hunting into a pipeline you can manage instead of a pile of tabs and half-finished applications.

## What this is for

Use this when you want one board for:

- saved jobs
- active applications
- interview stages
- follow-ups

## Workflow

### 1. Turn alerts on before you start applying

Create LinkedIn job alerts for:

- role
- location
- seniority
- key companies if needed

### 2. Save jobs into a real tracker

Each job should move through stages like:

- Saved
- Screening
- Applied
- Interviewing
- Waiting
- Closed

### 3. Review the board daily or every other day

Use the board to decide:

- what to apply to
- what to archive
- who needs follow-up

### 4. Use the model for triage, not blind mass applying

```text
Review these saved roles and tell me:
- which are strongest matches
- which need custom applications
- which I should skip
- what evidence I already have for each one
```

## Failure modes

- saving jobs with no stage tracking
- applying without a follow-up column
- too many alerts with no filter logic
