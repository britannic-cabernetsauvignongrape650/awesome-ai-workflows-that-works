# Code of Conduct

This project aims to be practical, friendly, and welcoming.

## Our standard

Please:

- be respectful and constructive
- assume good intent and discuss ideas in good faith
- give feedback on content, clarity, and usefulness
- help keep the repository welcoming to newcomers

## Unacceptable behavior

This includes:

- harassment or personal attacks
- insulting, hostile, or exclusionary language
- bad-faith contributions or spam
- publishing private information without permission

## Scope

This code of conduct applies to issues, pull requests, discussions, and other community interactions related to this repository.

## Enforcement

Maintainers may edit, remove, or reject contributions that do not follow this standard.

If needed, open an issue describing the problem and the maintainers can review it.
