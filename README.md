## 🧰 Troubleshooting

If a workflow does not run:

1. Check that you extracted all files.
2. Make sure the app you need is installed.
3. Confirm you opened the right guide.
4. Review the sample input format.
5. Try a smaller test case.
6. Restart the app and try again.
7. Compare your setup with the steps in the repo.

If local AI feels slow, close other apps and try a smaller model. If a prompt gives weak results, shorten the task and add more detail.
